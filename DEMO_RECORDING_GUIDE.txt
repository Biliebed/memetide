OPTION 1: Simple Browser Recording (Easiest - 30 min total)

Step 1: Open Demo Page (2 min)
  → Open: ~/memetide/demo.html in browser
  OR
  → Open: https://memetide-production.up.railway.app/docs

Step 2: Install Screen Recorder (3 min)
  → Mac: Use QuickTime (built-in)
  → Windows: Use Xbox Game Bar (Win + G)
  → Any OS: Install OBS Studio (obsproject.com)
  OR
  → Use Loom.com (browser-based, easiest)

Step 3: Record (10 min)
  1. Start recording
  2. Show demo.html page - click all 3 buttons
  3. OR show /docs - run scan, multichain, auth
  4. Open console - run WebSocket code
  5. Stop recording (aim for 60-90 seconds)

Step 4: Add Voiceover (10 min)
  → Use Audacity (free) or phone voice recorder
  → Read script from DEMO_VIDEO_SCRIPT.md
  → Overlay audio in video editor

Step 5: Export (5 min)
  → Format: MP4, 1080p, 30fps
  → Keep under 90 seconds
  → Keep under 100MB file size
